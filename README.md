# 🐍 My First Python Data Ingestion Script

## 📝 Project Description
This is my very first Python project! I built a simple data ingestion script that interacts with a user via the terminal, collects their profile information, and automatically saves that data into a local text file (`user_data.txt`). 

This project helps me understand how Python handles user inputs, variables, and basic file operations (reading and writing data), which are foundational concepts for Data Engineering.

## 🛠️ Tech Stack
- **Language:** Python 3
- **Concepts Used:** Input/Output operations, File Handling, String Formatting (f-strings).

## 🚀 How to Run It
1. Make sure you have Python installed.
2. Run the file using the command: `python main.py`
3. Enter your details when prompted, and check the generated `user_data.txt` file!
